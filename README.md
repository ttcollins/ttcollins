## 🗂 Selected Projects

### [SpringBootUtils](https://github.com/ttcollins/SpringBootUtils)
> A multi-module Spring Boot reference suite covering common enterprise patterns and real-world use cases.

**Tech:** Java · Spring Boot · JUnit 5 · Mockito · Flyway · Hibernate Spatial · Spring Batch · Spring Retry · PostgreSQL · Maven

| Module | What it demonstrates |
|---|---|
| `SpringBootTesting` | Unit, integration, and end-to-end testing patterns with JUnit 5 & Mockito |
| `flyway` | Database schema versioning and migration with Flyway |
| `hibernatespatial` | Geospatial queries using Hibernate Spatial |
| `springbatch` | Batch job processing with Spring Batch |
| `springretry` | Resilience and retry logic with Spring Retry |
| `todoapp` | Full CRUD REST API |
| `formulasandfunctions` | Business logic formulas and utility functions |

Each module is independently buildable with its own README. This repo serves as both a personal learning record and a practical reference for Spring Boot patterns.
